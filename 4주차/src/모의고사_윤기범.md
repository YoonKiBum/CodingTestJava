### 문제 접근 방식
  - 각 사람별 반복을 확인하였음
    - 1번 1, 2, 3, 4, 5 반복 
    - 2번 2, 1, 2, 3, 2, 4, 2, 5 반복
    - 3번 3, 3, 1, 1, 2, 2, 4, 4, 5, 5 반복
    
  - 문제의 수가 각 사람별 반복횟수보다 많은 경우 % 를 사용하여 해결
 ```Java
if (temp > array.length-1) {
    temp %= array.length;
}
```
 - tempArr 기준에 맞추어 정렬 
 - 동점자가 여러명이면 번호순으로 오름차순 정렬
 
### 주의할 점
  - % 이용하여 런타임 에러 방지
